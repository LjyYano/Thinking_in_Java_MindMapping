## 学习路线建议

如果你是 Redis 初学者，建议按照以下顺序阅读：

1. **基础入门**：先阅读"环境配置"搭建开发环境
2. **数据结构**：依次学习 SDS、Dict、Sorted Set、ziplist、quicklist 等数据结构（文章 2-6）
3. **运行机制**：理解 main 函数启动、epoll、Reactor 模型、事件循环（文章 7-10）
4. **多线程与命令**：学习多线程 IO 和命令处理流程（文章 11-12）
5. **持久化**：掌握 RDB 和 AOF 持久化机制（文章 13-15）
6. **客户端服务器**：了解客户端和服务器的实现（文章 16-17）
7. **高级特性**：学习复制和哨兵机制（文章 18）
8. **最新特性**：查看 Redis 7.0 新特性和内存优化技巧（文章 19-20）
