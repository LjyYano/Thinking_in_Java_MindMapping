异度之刃 1 作为我的 jrpg 启蒙之作，在很大程度上影响了我的游戏美学以及生活中的一些方面，例如我写的第一篇文就是以异度宇宙作为蓝本来描绘的。在我心目中异度之刃能与塞尔达传说相媲美（因为时之笛是我小时候玩的第一款游戏，这个系列我是玩了将近十多年的）。恰好最近游戏荒，打算重刷一遍异度 2，为明年一代重置最好准备，在此写一篇个人对于异度之刃系列的粗略理解，本文将梳理一遍异度之刃 2 的大致剧情流程，介绍一下部分背景设定，这也是第一次在站投稿专栏，做得不好的地方和错误之处地方希望能够指出，也感谢所有能看这篇文的人。另外有兴趣的玩家可以去油管上看 miiks 大神的解析。

在开始主题之前首先明确几个概念：

# Gate/佐哈尔

`佐哈尔` (Zohar) 一词源自犹太教卡巴拉密教文献，译为光辉，其中记载有对“`至高神`”的解释。至高神，即“波动存在”，“他”不是以肉体的形式存在于我们的宇宙中，而是属于更高次元的宇宙，“他”和现实宇宙的联系便是佐哈尔。佐哈尔是`上纬宇宙`在`下纬宇宙`的意志体现，因为本身就是属于高纬宇宙，所以可以掌控下纬宇宙的一切。而我们所在的宇宙可以解释为在时间垂直纵轴上一个个平行的事象平面，佐哈尔可以回应能动主体—人类的愿望，以操纵时间轴倾斜的方式增大该“未来事项”的可能性，从而获得几近无限的能源。

Gate，门，贯穿异度之刃宇宙，其意义就如同佐哈尔之于异度传说，二者拥有及其相近的能力。异度之刃宇宙中的 gate 在游戏中的体现是可以连接异度 1 和 2 世界的装置（`下位宇宙分为物质宇宙和精神宇宙`，两个宇宙相互为表里，互不交叉，互不干涉，而 gate 提供了这种干涉的可能性，而后果就是宇宙的毁灭），拥有无限的能源，是莫纳德、天之圣杯、永劫、乐园等一切的驱动器，而其能源则是通过回应使用者的愿望从而获得，即“`相转移性`”。由于 gate 的官方描述实在是太少，没有任何官方设定证明 gate 就是佐哈尔 (ps：避免侵权，因为异度装甲的版权是 SE 的而异度传说则是南梦宫的），但是为了方便理解很大程度上我个人会把他等同于佐哈尔 (ps: 据说破解解包后 gate 的注释就是佐哈尔，但并没有写在官方的设定集里）。

值得注意的是，无论是一代的巨神扎赞，还是二代的半身克劳乌斯，都只是沾染了 gate 力量的人类，只是伪神而已。

# 三位一体

三大本体这一概念源自新柏拉图主义，三大本体即`太一`，`理智`，`灵魂`，对应在异度神剑 2 中用于管理 gate 的三大核心处理器。普罗提诺论证说太一、理智和灵魂为“三个首要本体”。太一，是至高，能动的原因，为“本质“，即神。它不具备多样性，是不可分割的原初的“一”。而本质在克劳乌斯的启动 gate 后进入异度神剑的世界，成为了“波动存在”，即“莫纳德”，自命“阿尔维斯”, 作为世界的观察者和引导者，是一代中真正的神。而理智和灵魂则留在异度之刃 2 的世界成为了天之圣杯灭和光，值得一提的是，光觉醒后胸口的大文字是希腊文的“灵魂”，而灭的武器则是莫纳德的模样，灵魂的第三把剑也是形同莫纳德，或许三个核心处理器之间存在部分同调性的可能。

根据普罗提诺论证，太一产生理智，理智产生灵魂，是否三个核心处理器存在性能上的差异呢？阿尔维斯已经波动化，显然是性能最强的一个（一代结局的创世和能够预知无限遥远的未来，并且拥有旧世界的记忆），但灭并没有真正的觉醒，是否觉醒的灭实力强于光？这也是一个未知数。但灭不知道永劫的存在，而光则是永劫的主控者，有了这一大杀器，实力 自然也不好判断，或者灭和光修炼个几万年，也可以成为阿尔维斯那样的波动存在？当然这一段只是我个人的心血来潮时的臆想。

# 剧情正片

克劳乌斯的回忆也正是故事的一开始，反政府军已经攻上了“豆茎”，即将占领“乐园”，这时候克劳乌斯接管了“乐园的”全部控制权，试图启动 gate 创造全新的世界，他觉得 gate 是神赐予人类的礼物，gate 的力量可以带着人类前往新的地平线， 而 gate 无限的能量极有可能被反政府军用于战争，克劳乌斯并不想 “把它交给那群家伙”，他从心底是对人类感到绝望的，于是他启动了 gate，gate 回应了他的愿望“希望人类变得完美，不再愚蠢不再有战争”并开始调整时间轴朝着这个可能的未来方向进行。然而这个可能性为零，人类终究不是完美的生物，但此时间轴已经拨动无法到达未来也无法返回过去，于是世界崩坏，异度 2 与异度 1 的世界相互连接，克劳乌斯的半身和加拉提亚的半身穿越到异度 1 的世界成为了巨神扎赞和机神美纳斯。而留在异度 2 世界的克拉乌斯成为了第十章所见的神，加拉提亚成为了世界树底的异形王。

为了赎罪，也为了期待人类能够更进一步，克劳乌斯的半身开始了他的创世计划，他创造了能够重组旧世界的物质的分子体，也就是云海。随后将记载着生命信息的源物质即核心水晶投入云海，创造出生命，最初的生命体进化成巨神兽，从而诞生了新人类（类似于一代巨神界，霍姆斯，人类的关系）。但是克劳乌斯会担心新人类中有和他一样狂妄，愚蠢又懦弱的存在，他开始了第二个计划——`异刃系统`（异刃系统原本是为了追求永生让核心水晶取代脑细胞的产物，莫尔斯之地的异形就是旧世界人与核心水晶结合的产物）。所谓的异刃会将外界的信息和御刃者的情报发送给中央处理器即两个天之圣杯，而异刃则会受到来自他们的御刃者的影响从而产生稀有异刃，因此在游戏中同调异刃不同的心相有加成就是这个原因。天之圣杯则会储存下异刃传输过来的信息并返还新的`进化代码`，随着进化代码的累计，`异刃最终会进化成新的巨神兽`。克劳乌斯通过执行这个计划便可以解决人类的存续和观察人类两项实验，不得不感叹科技宅的本事啊。

而经过了漫长的观察，克劳乌斯得到了答案，那就是新人类和他并没有区别，一样的狂妄，愚蠢又懦弱，于是在他的“默许”之下，爬上世界树的`麦佩尼`带走了理性和灵魂两枚处理器。

麦佩尼幼年丧母，虽然手刃了仇人却也为此感到罪恶，也因此加入教廷希望救济世人偿还罪孽。随着麦佩尼逐渐走向教廷的上层，就越是对于利用信仰控制人民的教廷上层和被教廷蒙骗的民众感到厌恶，直到他看到自己亲手拯救的士兵杀人抢劫的样子，他终于彻底对这个世界绝望：“神啊，这就是你所期望的世界么”，并且 cg 里虽然很隐晦，但可以确定，麦佩尼确实将那个救下来的婴儿给掐死了。

为了证实他的想法，他爬上了世界树，希望得到神的指引，从他没见到神而仓惶逃走来看，克劳乌斯应该也是给他看了那场幻境，让他直面了内心的恐惧和丑陋。他只得以拿走了光灭离开，却也坚定了黑化的信念——这丑恶世界的一切都没有价值，应当抹消。回到幽界后，见识到了灭的力量，又从斯塔尼夫那了解到了异刃，巨神兽的真相，旧世界的部分真相后，认为自己看见的幻境就是神给自己的“神谕”，是神告诉他人类就是如此丑陋。

此时受麦佩尼影响的灭觉得自己已经了解了人类——人类总是在寻死，而自己则是神派来在人类的背后“轻轻推上一把”的存在，但他又对自己的御刃者麦佩尼敢想不敢为的行为嗤之以鼻，毕竟麦佩尼也是一个寻死的人类。随后他决定离开麦佩尼自己去毁灭世界，索性他没有一开始就着手毁灭世界，而是四处闲逛，本着游戏的心态，这也给了幽界喘息的机会。

灭走了，但是麦佩尼却急了，他觉得自己是离神最近的人，是“神的意志”，天之圣杯是“手段”，“神谕”必须由“神的意志”来执行，“手段”离开了自己是不行的。不然，自己精神的寄托就消失了，自己的存在就没有了“意义”，也是他圣杯大战后急于消灭反抗军（`阿德尔`，光）和五百年后雷克斯一行人登上世界树而急躁的原因。于是他将光交给了阿德尔，让阿德尔去阻止灭，并秘密进行着核心水晶移植的人体实验。

随后的圣杯大战中，由于阿德尔畏惧着`天之圣杯`的力量，并不能完全的引导光的力量导致光并不能真正的觉醒并且暴走（PS: 失控中的光链接上了 gate，引动了 gate 的力量可以通过“未来式”看到五百年后的因果，这也是为什么一开始焰愿意将一半的生命共享给雷克斯。由此我也推测也许旧世界大战中，永劫才是旧人类利用 gate 力量的最终目标，用核心处理器来作为控制永劫和 gate 的意识），随后盛极的伊拉巨神兽陨落，伊拉亡国，斯佩比亚的皇帝战死，米托尔为了保护佐田彦牺牲，光深受刺激因此封闭自我制造了焰人格，阿德尔前往灵洞封印了第三把剑，并封印了焰（焰温柔，知性，恰恰一切正好是光所没有的，甚至可以说焰就是阿德尔所期望光的样子），随后不知所踪。灭战败并且核心水晶受损失去了天之圣杯的力量，由此开始长达五百年的旅行。劳拉在教廷的追杀战死去，真成为食人种，芳和佐田彦被教廷俘虏成为食刃种技术的实验体。黄金之国剧末这一刻开始所有人都在朝着五百年后的因果相前进，虽不同道却殊途同归。

随后的五百年中，伊拉残余的王室（即王弟一派）偶然间得到了海蛇的控制核心`神圣锁链`，为了掌握人心自诩`英雄阿德尔后裔`，成立洛修利亚王国并为了避免他国干涉将玄武沉入云海中，由于围剿反抗军那些龌龊把柄捏在阿卡迪亚的手里，因此洛修利亚被迫每年向教廷供奉核心水晶。另一边麦佩尼逐渐清理掉了教廷高层成员，成为了教皇（ps: 五百年前阿卡狄亚的教义为“死亡并且结束，灵魂终将得到救赎”，而麦佩尼登基后教义却变为了“犯错不是结束，活着就能赎罪”）并开始了他的灭世的计划，且为了防范天之圣杯的再次出现，一边为自己移植了芳的核心水晶，一边控制对其他国家的核心水晶也就是`洗礼`，因为核心水晶的本质就是收集生命信息积累进化代码，所谓的洗礼就是抽走其中的进化代码，被洗礼过的核心水晶固然可以提高同调几率却也`永远无法进化成巨神兽`，被抽走的进化代码则被炼制成“神铠”，这也是五百年来再也没有巨神兽诞生和资源越来越匮乏的主要原因。

与此同时真四处流浪，背负着铭记劳拉和所有朋友而迷茫的活着，一边思考着自己以及异刃存在的意义——异刃真的只是作为人类的附属品，为人类而活着吗？虽然拥有近乎永生的寿命，却无法留存记忆，即使在生命结束后也只是等待下一次同调，可再次觉醒的他还是原来的他么。此时真遇到了同样迷茫的灭，在这次的漫长旅途中，灭对于曾经深信不疑的“使命”产生了怀疑——这真的是自己的意志吗？还是麦佩尼或者神灌输给他的意志？这又变成了科幻小说的经典哲学思辨——人工所创造出拥有自我思想的个体，其思想到底属于自己还是属于创造者？灭就这样在迷茫中着寻找答案，直到与真的再次相遇，真是他见过没有自己愿望的存在。于是结合双方的经历，灭为真指出了新的道路——去问问神，自己存在的意义，既然世界的把异刃当做人类的工具，那就破坏掉这个错误的世界。真于是接受了灭指出的道路，并将其作为自己活着的一个意义，但是真始终有着自己的坚持，即便走上了灭世的道路，他也没有滥用食人种技术去扩张势力，这个世界上依旧会有和劳拉一样的御刃者，也有和曾经的他一样的异刃，他没有权利去替别人做出选择，强加的“自由”只会导致悲剧。收集到的核心水晶也没有去同调，他不希望有新的异刃来到这个世界上受苦。但是他要去弑神，去破坏这个世界不合理的规则，但是真也明白这同样是一种强迫，劳拉、以前在伊拉的记忆，使他这条道路隐隐地有一种抗拒，他渴望听到一个更好的答案，所以在多出剧情里出现了真因为摇摆不定而放水的刻画，甚至在第六章阐明对雷克斯有一丝期待，雷克斯也在第七章中说真的眼中满是悲伤。

在最后的世界树之战中，佐田彦连同伊拉的飞船替雷克斯一行人挡住了教廷的搏命一击，正如他所说，有的人死不了也无论如何不能死，五百年前米托尔替他挡了一击，他背负着仇恨和挚友的回忆熬了五百年，即便他加入了伊拉参与了灭世但他还是一个人类，有着人类的立场，五百年后的此刻他将这份传承的使命交给了雷克斯一行人，他自己带着他姐姐劳拉的遗体壮烈赴死（ps: 虽然二周目可以在世界树底层拉他入队，但二周目属于拓展，只为游戏服务并不是补充剧情）。真则在第七章的光觉醒之战后，也开始相信了雷克斯所展现的可能，随之与灭告别，选择留下来等待雷克斯的到来，在与雷克斯的一战中，真愿意再一次相信异刃与人类的可能性，此刻真选择了怀抱过去，微笑着将未来托付给了雷克斯。

此时的灭找到了克劳乌斯，问出了一直想要的问题，克劳乌斯并没有给他明确的答案，而是告诉他这是“你们”的意志。这个回答并没有让灭满意——既然和麦佩尼一样都是灭世的想法，怎么证明这其中有自己的意志呢？那只能问问那个小子了，最终他又一次战败了，也承认了雷克斯的可能性，并对此感到高兴——麦佩尼是不会认可雷克斯的，这只能是他自己的意志。他赢了，毁灭世界；败了，雷克斯会继续前进，旧的残酷世界也就不复存在了，而自己是残酷世界的一份子也被消灭。如果连他都无法打败，还谈何未来？那么他也通过自己见证了未来，他也是因此存在过的证明，也欣然接受了麦佩尼作为自己御刃者这个事实。

故事的最终章，克劳乌斯使用 gate 最后的力量来聚集巨神兽，让一切回归“乐园”（ps：最后围绕世界树的那些文字译为“回到天堂”，灵魂胸口的拉丁文字译为“但義人的路好像黎明的光，越照越明，直到日午”)，与此同时，克劳乌斯身边的黑洞传来`一代主角修尔克`的名言：“我们要凭借自己的力量斩杀神，开辟未来！”此刻，同时雷克斯和修尔克各自行至命运的终点，克劳乌斯的愿望也达成，gate 从这片宇宙消失，两个世界形成闭环。

# 写在最后

从 2 代入坑的玩家来说，似乎这只是个抽卡开后宫，王道剧情的故事。但事实上，整个异度系列是高桥个人结合宗教，哲学，科学，心理学等一系列知识，跨越万年的故事，将高桥自身对于人与自然，人与人，人与自身的拷问、感悟糅杂其中，有的人过于执着的将自身的意志强加于世界，因此陷入黑暗之中迷失自我，这是人性的缺陷。而“神”整个概念贯穿整个系列，所有人在茫茫的宇宙和有限的生命历程中，寻找自己的真正渴望和向往的神。或许这也是异度之刃吸引我的最大原因，在当下 jrpg 已经很难在内容上再做突破，一味的追求精美的画质表现反而会落了下乘，异度系列最大的闪光点就是思考，这已经不是画质和剧本能赋予游戏的内容了，游戏是最能体现第九大艺术的形式，而异度系列就是在这个方向上努力做出突破，努力在游戏和艺术上划等号。

期待明年一代的重置，以及未来的续作，并再次谢谢阅读本文，如有错误感谢指正。
